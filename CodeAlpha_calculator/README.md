# CodeAlpha Calculator

This is a basic calculator project made using HTML, CSS and JavaScript.

## Features
- Addition, Subtraction, Multiplication, Division
- Delete (DEL) button
- Clear (AC) button
- Keyboard support
- Mobile responsive design
- Premium dark UI

## Technologies Used
- HTML
- CSS
- JavaScript
