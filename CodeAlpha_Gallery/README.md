# CodeAlpha Gallery

[Live Demo](https://vedantb3008.github.io/CodeAlpha_Tasks/CodeAlpha_Gallery/)

A premium image gallery project showcasing nature, cityscapes, animals, humans, and monuments. Built with HTML, CSS, and JavaScript, it provides a clean, responsive, and visually appealing interface.

## Features
- Category filters: Nature, City, Animals, Humans, Monuments
- Responsive grid layout for different screen sizes
- High-quality sample images (via Picsum)
- Smooth navigation with previous/next controls
- Premium dark-themed UI

## Technologies Used
- HTML
- CSS
- JavaScript

This project is created as part of the CodeAlpha Internship program.
